# Vue Mastery Beginner Path
| This repository is for educational purposes only
## Project setup
```
npm install
```

### Mock json database 
```
npm i -g json-server
json-server -d 1500 db.json
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
